## What it is

**Agent Dev-Sprint: Build, Deploy, and Battle** — a high-velocity, hands-on
one-day event for developers, engineers, and AI builders to master autonomous
agent architectures and put their code to the test in a live, gamified arena.
Bridges raw theory and production-ready execution: orchestrating LLM inference,
implementing advanced tool-use, and keeping agents from looping endlessly.

## The event flow ("Laps")

1. **Lap 1 — The Warm-Up (Technical Deep Dives):** core pillars of agent
   architecture — Perception, Reasoning (ReAct / Chain-of-Thought), Tool-Calling.
2. **Lap 2 — The Straightaway (Hacking Phase 1):** set up the environment from
   starter kits, build core logic, test local execution loops.
3. **The Pit Stop (Lunch & Mentorship):** refuel, network, unblock with mentors.
4. **Lap 3 — The Final Corner (Optimization):** fine-tune prompts, integrate
   custom tools, run sandbox simulations to prep for the tournament.
5. **The Checkered Flag (Arena Showdown):** deploy your agent endpoint to the
   live leaderboard; agents battle head-to-head in a gamified environment.

## The Agent Arena (grand finale)

Custom-built autonomous agents are unleashed into a live, gamified simulation.
They must **parse dynamic data, use APIs to make strategic decisions, and react
to real-time events** to climb the leaderboard.

## Mentorship

Direct access throughout the day to industry experts in AI and solutions
architecture — to debug reasoning chains, optimize context windows, and sharpen
the agent's decision-making.
